Prerequisites

  • Python 3.12+
  • Windows (prototype), with terminal access

Run the Terminal

From the project directory:

> Terminal_Launcher.bat

This launches terminal.py, displays the ASCII logo, and opens the interactive shell.

Folders

  • ROOT/*: engine defaults (read-only)
  • User/*: project-specific content (overrides, writes)
  • Sigils/, Relations/, Scripts/, Commands/, Modules/
  • ROOT/Utilities/: Windows tools (.exe/.bat/.cmd/.ps1/.py)
  • User/Logs/: runtime pulse and change logs
